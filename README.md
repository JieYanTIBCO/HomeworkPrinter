# HomeworkPrinter

作业打印系统，支持生成多种类型的作业PDF文件

## 功能特性
- 数学作业生成
- 语文作业生成  
- 英语作业生成
- 支持自定义模板
- 自动编号
- 多页面支持

## 目录结构
```
HomeworkPrinter/
├── config/         # 配置文件
├── templates/      # 模板文件
├── output/         # 输出文件
├── src/            # 源代码
│   ├── math/       # 数学作业
│   ├── english/    # 英语作业
│   └── chinese/    # 语文作业
└── tests/          # 测试代码
```

## 使用说明
1. 配置config/settings.yaml
2. 运行src/main.py
